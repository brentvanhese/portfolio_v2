Opdracht

Je maakt een site over een thema (hobby, interesse, idool, favoriete film, favoriete artiest, boek, vereniging, huisdier, game …)
Deze themasite is met Bootstrap gemaakt.
De richtlijnen voor deze site zijn als volgt:

  • minimum 3 pagina’s,
  • er is navigatie,
  • je maakt veelvuldig gebruik van grafische elementen,
  • geen placeholder content (‘lorem ipsum’), eventueel kopieer tekst van Wikipedia of andere open sites (met bronvermelding),
  • de site voldoet aan de richtlijnen ivm design en UX uit de cursus user interfaces,
  • de site responsive.
